# Desafio de Algoritmos - SENAI 🎓⚙️

Este repositório contém a entrega de um projeto/desafio proposto no curso do **SENAI**. O objetivo principal do exercício é treinar a **lógica de programação** através da criação de algoritmos para situações cotidianas.

Para cada uma das situações propostas, foram desenvolvidas duas representações:
1.  **Fluxograma:** Representação visual passo a passo (em `.png`).
2.  **Pseudo-código:** Descrição textual estruturada da lógica (em `.docx`).

---

## 📋 Situações Desenvolvidas

Abaixo estão as 5 atividades do dia a dia mapeadas neste projeto:

1.  💼 **Ir ao trabalho**
    * `1_trabalho_fluxo.png`
    * `1_trabalho_pseudo.docx`
2.  🚗 **Troca de um pneu de carro**
    * `2_pneu_fluxo.png`
    * `2_pneu_pseudo.docx`
3.  💡 **Troca de uma lâmpada**
    * `3_lâmpada_fluxo.png`
    * `3_lâmpada_pseudo.docx`
4.  🚌 **Pegar um ônibus**
    * `4_ônibus_fluxo.png`
    * `4_ônibus_pseudo.docx`
5.  🍰 **Preparar um bolo simples**
    * `5_bolo_fluxo.png`
    * `5_bolo_pseudo.docx`

---

## 🛠️ Tecnologias e Conceitos Aplicados

* **Lógica de Programação:** Estruturação de sequências de passos lógicos, condições e repetições necessárias para concluir uma tarefa.
* **Fluxogramas:** Criação de diagramas para visualização de processos.
* **Pseudo-código:** Escrita de algoritmos em linguagem natural estruturada (Portugol/Lógica pura).
